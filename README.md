# Yorfiles 

These are my i3wm configurations specifically for archlinux or more advanced distributions (for example voidl) etc.

<hr>

#### Dependencies to install

* ArchLinux (*Optional*)
* i3wm
* xfce4-panel
* rofi
* picom
* pcmanfm
* kitty
* ranger
* iceweasel-broser
* nano (*Text Editor*)

<hr>

#### Themes

* Deepin Icon Theme
* Andromeda Theme
* Mendlo & Victor Mono
* Minimalist Wallpaper 
* Gogh Themes for Terminal
